# Dental Clinic Website

## Project Overview
Polish dental clinic website — modern, warm, and professional.

## Stack
- HTML5
- Tailwind CSS (via CDN only)
- Vanilla JavaScript
- Custom CSS (style.css)

## Brand Colors
| Name | Hex |
|------|-----|
| Navy Blue | `#1B2D5B` |
| Orange | `#E8601C` |
| White | `#FFFFFF` |
| Light Gray (background) | `#F8F9FA` |

## Typography
- **Headings**: "Plus Jakarta Sans" (weights: 400, 600, 700) — Google Fonts
- **Body**: "Inter" (weights: 400, 500) — Google Fonts

## Design Style
- Modern minimalist, warm medical feel
- NOT cold or clinical
- Dominant shapes: circles and rounded corners (border-radius heavy)
- Consistent use of soft shadows and ample whitespace

## Rules
- DO NOT use any CSS framework other than Tailwind
- DO NOT add dark mode
- All sections must be fully mobile responsive
- Image areas: use placeholder `<div>` elements with `bg-gray-200` and appropriate aspect ratios — no real images yet
- Language: Polish (`lang="pl"`)
